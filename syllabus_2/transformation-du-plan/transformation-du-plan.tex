\subsection{La translation}
Une translation est une transformation du plan qui déplace tout point~:

\begin{itemize}[label=\faIcon{angle-right}]
    \item dans une \emph{même direction},
    \item dans un \emph{même sens},
    \item et de \emph{même distance} que la longueur du vecteur.
\end{itemize}
L'élément caractéristique d'une translation est un segment orienté (symbolisé
par une flèche) appelée \emph{vecteur}.

Exemple de notation : \(t_{\overrightarrow{GH}}(A,B,C,D,E) = A',B',C',D',E'\)

\subsection{La symétrie orthogonale}
Une symétrie orthogonale est une transformation du plan qui envoie tout
point~:
\begin{itemize}[label=\faIcon{angle-right}]
    \item de \emph{l'autre côté} de l'axe,
    \item sur la droite \emph{perpendiculaire} à l'axe passant par ce point,
    \item et à une \emph{même distance} de l'axe.
\end{itemize}
L'élément caractéristique d'une symétrie orthogonale est une droite
appelée \emph{axe.}

Exemple de notation : \(S_{a}\ (A,B,C,D,E) = \ A',B',C',D',E'\)

\subsection{La symétrie centrale}
Une symétrie centrale est une transformation du plan qui déplace tout
point~:
\begin{itemize}[label=\faIcon{angle-right}]
    \item de \emph{l'autre côté} du centre,
    \item sur la droite passant par le point et le centre,
    \item et à une \emph{même distance} du centre.
\end{itemize}

L'élément caractéristique d'une symétrie centrale est un point appelé
\emph{centre de symétrie}.

Exemple de notation :\(S_{O}\ (A,B,C,D,E) = \ A',B',C',D',E'\)

\subsection{La rotation}
Une rotation est une transformation du plan qui déplace tout point~:

\begin{itemize}[label=\faIcon{angle-right}]
    \item à la \emph{même distance} par rapport au centre
    \item selon \emph{une amplitude} donnée
    \item selon \emph{un sens} donné
\end{itemize}

Les éléments caractéristiques d'une rotation sont
\emph{un centre, une amplitude} et \emph{un sens}.

Le sens est donné par le signe de l'amplitude. S'il est positif, on tourne dans
le sens anti-horloger. S'il est négatif, on tourne dans le sens horloger.
\begin{center}
    \begin{tikzpicture}[>=latex, -latex]
        \tkzDefPoints{0/0/O}
        \tkzDefPoint(100:1){A}
        \tkzDefPoint(80:1){B}
        \tkzDrawArc[->, black](O,A)(B)
        \tkzLabelPoint[anchor=center](O){$+$}
        \tkzDefPoint(90:1){Label}
        \tkzLabelPoint[above](Label){sens anti-horloger}
    \end{tikzpicture}
    \hfil
    \begin{tikzpicture}[>=latex, -latex]
        \tkzDefPoints{0/0/O}
        \tkzDefPoint(100:1){A}
        \tkzDefPoint(80:1){B}
        \tkzDrawArc[<-, black](O,A)(B)
        \tkzLabelPoint[anchor=center](O){$-$}
        \tkzDefPoint(90:1){Label}
        \tkzLabelPoint[above](Label){sens horloger}
    \end{tikzpicture}
\end{center}

Exemple de notation : \(r_{A;- 45\degres}\left(\lbrack BC \rbrack\right) = \lbrack B'C'\rbrack \)
