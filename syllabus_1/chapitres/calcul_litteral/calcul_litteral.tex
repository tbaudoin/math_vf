\section{Valeurs numériques d'une expression algébrique}

Calculer la valeur numérique d'une expression, c'est trouver le résultat d'une expression algébrique pour une valeur de partie littérale donnée. Prenons un exemple :
\begin{equation*}
    8a+5
\end{equation*}
Calculer sa valeur numérique pour \(a = 2\) revient à remplacer \(a\) par \(2\) dans l'égalité de départ et à respecter les priorités des opérations pour calculer le résultat.

Ici, cela nous donne~: 
\begin{center}
    $\begin{aligned}
    8\ .\ 2 + 5\  &= 16 + 5 \\
    &= 21
  \end{aligned}$
\end{center}

  \exemple
  Calcule la valeur numérique de l'expression \(3t + 5t^{2}\) si \(t = 6\).
  \vspace{4cm}

  \begin{exercise}
    Calcule la valeur numérique de \(x^{2} + 3x - 10\) si tu sais que \(x = - 3\).
    \vspace{4cm}
  \end{exercise}

Comment réduire une somme algébrique? 

Pour additionner deux termes semblables, il suffit de :
\begin{itemize}
    \item additionner les coefficients 
    \item conserver la partie littérale
\end{itemize}

Comment réduire un produit algébrique? 

Pour réduire un produit algébrique, il suffit de :
\begin{itemize}
    \item multiplier les coefficients entre eux
    \item multiplier les facteurs littéraux entre eux.
\end{itemize}

\exemples
\begin{enumerate}[cols=2, itemsep=1cm, label=]
    \item \(3a\cdot 5a=\)
    \item \(7a \cdot (-3b)=\)
    \item \(a\cdot a\cdot a\cdot a=\)
    \item \(-5x\cdot 3y=\)
\end{enumerate}

        \item \(-8a\cdot 2a=\)
        \item \(-4x^2y \cdot (-2x)=\)

\end{exercise}
