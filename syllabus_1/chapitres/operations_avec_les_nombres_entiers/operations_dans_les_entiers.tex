Pour multiplier deux entiers :

\textbf{Si les entiers sont de même signe,} \dotsforanswer{3}
\exemples
\begin{itemize}[cols=2, itemsep=1cm, label=]
    \item $(-3)\cdot (-6)=$
    \item $7\cdot 6=$
    \item $-2\cdot (-3)=$
    \item $8\cdot 7=$
\end{itemize}
\vspace{1cm}

    \item $(-3)\cdot 6=$

\end{itemize}

    \begin{enumerate}
        \item \(( - 4) \cdot a\) est toujours négatif.
